# 🚀 中国境内网站上线全流程操作指南

> 基于 chengnanblog.cn 的实际操作经验整理，适用于个人网站/博客在阿里云上线的完整流程。

## ① 购买阿里云 ECS + 域名

### 服务器（ECS）
1. 进入 [ECS 购买页](https://ecs-buy.aliyun.com/)
2. 推荐配置：2核2G、Ubuntu 22.04、40G 云盘
3. 安全组默认开放 **22（SSH）、80（HTTP）、443（HTTPS）** 端口

### 域名
1. 进入 [万网域名注册](https://wanwang.aliyun.com/)
2. 搜索并购买域名（`.cn` 需实名认证）

## ② 服务器部署

### 安装 Docker
```bash
ssh root@<服务器IP>
curl -fsSL https://get.docker.com | sh
systemctl enable docker && systemctl start docker
```

### 部署 Next.js 应用（以 Docker 为例）
```bash
# 拉取镜像并运行
docker pull <镜像地址>
docker run -d --name myblog --restart always -p 3000:3000 <镜像地址>
```

### 安装并配置 Nginx
```bash
apt update && apt install -y nginx

# 创建站点配置（先用 HTTP，备案期间通过 IP 访问）
cat > /etc/nginx/sites-available/mysite << 'EOF'
server {
    listen 80;
    server_name <你的域名>;
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
EOF

ln -s /etc/nginx/sites-available/mysite /etc/nginx/sites-enabled/
nginx -t && nginx -s reload
```

## ③ 域名解析

1. 进入 [云解析 DNS 控制台](https://dns.console.aliyun.com/)
2. 添加 A 记录：

| 主机记录 | 类型 | 线路 | 记录值 |
|---------|------|------|-------|
| `@` | A | 默认 | `<服务器公网IP>` |
| `www` | A | 默认 | `<服务器公网IP>` |

3. 等待 5-10 分钟生效，用 `ping 你的域名` 验证

## ④ ICP 备案

### 在哪操作
进入 [阿里云 ICP 备案系统](https://beian.aliyun.com/)

### 流程
1. **填写主体信息** — 身份证正反面、手机号、邮箱
2. **填写网站信息** — 网站名称、域名、服务器 IP
3. **上传资料** — 身份证照片、网站备案信息真实性承诺书
4. **阿里云初审** — 约 1 个工作日
5. **管局终审** — 约 7-20 个工作日
6. **短信核验** — 管局会发短信，按提示在 24 小时内验证

### 注意事项
- 备案期间域名**不能通过域名访问**，只能用 IP 访问
- 备案通过后，阿里云需要 **2-24 小时** 同步白名单
- 网站名称不能用纯英文，建议用中文名称

### 结果
备案通过后获得 ICP 备案号，格式如：`闽ICP备XXXXXXXXX号-1`

## ⑤ SSL 证书配置（HTTPS）

### 5.1 确认安全组
在 ECS 控制台 → 实例 → 安全组 → 确保 **443 端口** 已开放

### 5.2 申请免费证书
1. 阿里云搜索 **"SSL证书"** → 进入"数字证书管理服务"
2. 切换到 **"个人测试证书（原免费证书）"** 标签
3. 点击"创建证书" → 选"个人测试证书（免费版）" → 立即购买（0元）
4. 回到列表点 **"证书申请"**：
   - 域名填：`你的域名`（不带 www）
   - 验证方式：自动 DNS 验证
   - 填写联系人
5. 提交审核，等待签发（约 5-15 分钟）

### ⚠️ 常见坑：证书一直"审核中"
如果域名有**境外 CNAME 记录**（如 Vercel 双线部署），CA 机构从境外访问会走错线路，导致验证失败。
**解决方法**：临时暂停境外 CNAME → 等证书签发 → 再恢复。

### 5.3 下载证书
签发后点 **"下载"** → 选 **"Nginx"** → 解压得到 `.pem` 和 `.key` 文件

### 5.4 上传并部署
```bash
# 本地终端执行
ssh root@<IP> "mkdir -p /etc/nginx/ssl"
scp ~/Downloads/<域名>.pem root@<IP>:/etc/nginx/ssl/
scp ~/Downloads/<域名>.key root@<IP>:/etc/nginx/ssl/
```

### 5.5 更新 Nginx 配置
```bash
ssh root@<IP>

cat > /etc/nginx/sites-available/mysite << 'EOF'
server {
    listen 80;
    server_name <域名>;
    return 301 https://$host$request_uri;
}
server {
    listen 443 ssl http2;
    server_name <域名>;
    ssl_certificate /etc/nginx/ssl/<域名>.pem;
    ssl_certificate_key /etc/nginx/ssl/<域名>.key;
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
EOF

nginx -t && nginx -s reload
```

### 5.6 验证
浏览器访问 `https://你的域名`，看到 🔒 锁标志即成功

## ⑥ 公安备案

### 在哪操作
[全国互联网安全管理服务平台](https://beian.mps.gov.cn/)

### 流程
1. **注册账号** — 手机号注册 + 实名认证
2. **新增主体** — 开办主体管理 → 填写个人信息 + 身份证照片
3. **新办网站申请** — 主体审核通过后填写：
   - 网站名称：`<中文名称>`
   - 域名：`<你的域名>`
   - IP：`<服务器公网IP>`
   - 网络接入商：阿里云计算有限公司
   - 域名注册商：阿里云计算有限公司
   - 网站接入方式：租赁虚拟空间
   - 服务类型：博客/个人空间
   - 是否提供互联网交互服务：否（无评论功能时）
   - 涉及管制物品：否
   - 前置许可：否
   - 语种：中文简体
4. **等待审核** — 3-10 个工作日

### 需要准备的材料
- 身份证正反面照片
- 域名证书（阿里云 [域名控制台](https://dc.console.aliyun.com/) → 管理 → 域名证书下载）
- ICP 备案号
