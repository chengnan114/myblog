## ① 购买阿里云 ECS + 域名

### 服务器

进入 [ECS 购买页](https://ecs-buy.aliyun.com/)，推荐配置：

- **规格**：2核2G（个人博客足够）
- **系统**：Ubuntu 22.04
- **磁盘**：40G 云盘
- **安全组**：默认开放 22（SSH）、80（HTTP）、443（HTTPS）端口

### 域名

在 [万网](https://wanwang.aliyun.com/) 搜索并注册域名。`.cn` 域名需要实名认证，建议提前准备好身份证信息。

## ② 服务器部署

### 安装 Docker

```bash
ssh root@<服务器IP>
curl -fsSL https://get.docker.com | sh
systemctl enable docker && systemctl start docker
```

### 部署应用

以 Next.js 项目为例，构建 Docker 镜像并运行：

```bash
docker run -d --name myblog --restart always -p 3000:3000 <镜像地址>
```

### 配置 Nginx 反向代理

```bash
apt update && apt install -y nginx
```

创建站点配置（备案期间先用 HTTP）：

```nginx
server {
    listen 80;
    server_name <你的域名>;
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

启用配置：

```bash
ln -s /etc/nginx/sites-available/mysite /etc/nginx/sites-enabled/
nginx -t && nginx -s reload
```

## ③ 域名解析

进入 [云解析 DNS 控制台](https://dns.console.aliyun.com/)，添加以下记录：

| 主机记录 | 类型 | 记录值 |
|---------|------|-------|
| `@` | A | 服务器公网 IP |
| `www` | A | 服务器公网 IP |

等待约 5 分钟后用 `ping 你的域名` 验证是否生效。

## ④ ICP 备案

这是最耗时的环节，但也是中国境内网站的必经之路。

### 在哪操作

进入 [阿里云 ICP 备案系统](https://beian.aliyun.com/)

### 流程

1. **填写主体信息**：姓名、身份证、手机号、邮箱
2. **填写网站信息**：网站名称（必须用中文）、域名、服务器 IP
3. **上传资料**：身份证照片、备案真实性承诺书
4. **阿里云初审**：约 1 个工作日
5. **管局终审**：约 7-20 个工作日
6. **短信核验**：管局会发短信验证码，24 小时内完成

### ⚠️ 注意事项

- 备案期间域名**不能通过域名访问**，只能用 IP 直接访问
- 备案通过后，阿里云白名单系统需要 **2-24 小时**同步，期间可能仍会看到拦截页
- 网站名称不能用纯英文，必须有中文

备案通过后你会获得一个备案号，例如：`闽ICP备2026008150号-1`

## ⑤ SSL 证书配置（重点！）

这是我踩坑最多的环节。没有 SSL 证书，浏览器默认走 HTTPS 就会显示"无法访问"。

### 第一步：确认安全组

在 ECS 控制台确保 **443 端口已开放**！这一步很容易被遗忘。

### 第二步：申请免费证书

1. 阿里云搜索 **"SSL证书"** → 进入数字证书管理服务
2. 切换到"个人测试证书（原免费证书）"
3. 创建证书 → 选免费版 → 立即购买（0 元）
4. 证书申请：填写域名（**不带 www**），验证方式选自动 DNS 验证
5. 提交等待签发

### 🐛 踩坑记录：证书卡在"审核中"十几个小时

**原因**：我之前配置了 Vercel + 阿里云双线部署（国内走阿里云，境外走 Vercel）。DigiCert 的 CA 服务器在境外，走的是 Vercel 线路，导致域名验证一直失败。

**解决方法**：临时在 DNS 控制台**暂停境外 CNAME 记录** → 等几分钟 → 证书立刻签发成功。

> 教训：**申请 SSL 证书前，确保域名的所有线路都指向同一台服务器。**

### 第三步：下载并部署

签发后下载 Nginx 类型证书（`.pem` + `.key`），上传到服务器：

```bash
# 本地终端
ssh root@<IP> "mkdir -p /etc/nginx/ssl"
scp ~/Downloads/<域名>.pem root@<IP>:/etc/nginx/ssl/
scp ~/Downloads/<域名>.key root@<IP>:/etc/nginx/ssl/
```

### 第四步：更新 Nginx 配置

```nginx
server {
    listen 80;
    server_name <域名>;
    return 301 https://$host$request_uri;
}
server {
    listen 443 ssl http2;
    server_name <域名>;
    ssl_certificate /etc/nginx/ssl/<域名>.pem;
    ssl_certificate_key /etc/nginx/ssl/<域名>.key;
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

```bash
nginx -t && nginx -s reload
```

看到地址栏的 🔒 锁标志，HTTPS 就配置成功了！

> **提醒**：阿里云免费证书有效期 90 天，到期需要手动续期。建议在日历上设提醒。

## ⑥ 公安备案

ICP 备案通过后 **30 天内**需完成公安备案。不影响网站使用，但属于合规要求。

### 在哪操作

[全国互联网安全管理服务平台](https://beian.mps.gov.cn/)

### 需要准备的材料

- 身份证正反面照片
- 域名证书（阿里云域名控制台 → 管理 → 域名证书下载）
- ICP 备案号

### 关键字段填写参考

| 字段 | 值 |
|------|------|
| 网站访问地址 | `https://你的域名` |
| IP | 服务器公网 IP |
| 接入商所属地区 |  |
| 接入商名称 | 阿里云计算有限公司 |
| 网站接入方式 | 租赁虚拟空间 |
| 域名注册服务商 | 阿里云计算有限公司（原万网） |
| 服务类型 | 博客/个人空间 |
| 是否提供互联网交互服务 | 否（无评论时） |
| 涉及管制物品 | 否 |
| 前置许可 | 否 |
| 语种 | 中文简体 |

提交后等待 3-10 个工作日审核。
